# library
работа с файлами
Управление книгами - Простая инструкция

Что делает программа

· Добавлять книги
· Удалять книги
· Искать книги
· Показывать все книги
· Сохранять всё в файл

Как пользоваться

Запуск

python program.py
Основные команды

1 - Добавить книгу
2 - Удалить книгу  
3 - Очистить всё
4 - Найти книгу
5 - Показать все книги
6 - Выйти
Добавление книги

Программа спросит:

· Номер книги (например: 001)
· Название (например: "Гарри Поттер")
· Дата (например: 26.06.1997)

Поиск книги

Можно искать:

· По номеру
· По названию
· По дате

Удаление книги

Можно удалить:

· По номеру
· По названию

Как хранятся данные

Файл books.txt в формате:

001|Гарри Поттер|26.06.1997
002|Новая книга|01.01.2024
Важно

· Программа сама создаёт файл при первом запуске
· Русские буквы работают правильно
· Данные сохраняются автоматически
